Basemap + your data

Uses OpenStreetMap tiles as the default basemap. Load GeoJSON from a URL (must allow browser CORS) or from a file. For your SAM results.json (pixel polygons), set image size + geographic bounds to project onto the map.

Project results.json (pixel coords)

Segmentation lists [x,y,x,y,...] in image pixels. Map corners: south-west and north-east (WGS84).

Saved click points

Click Generate bounding box once; then every new or removed point updates the blue box automatically (until you clear all points). Yellow path = click order.
